## ChatGPT API integrtion on Laravel
#### **chat-gpt-integration** has the following listed features :)


- User Chat Interface
- Used Default SQLite Database.May use any Database Like MySQL or Mongodb

## _Requirements_
- PHP 8.3
- Laravel 11

## _Setup development_

>  _Step 1. Run this in your terminal :

````
    git clone https://github.com/gmanikandan-dev/chat-gpt-integration.git
````

>  _Step 2.

````
    cp .env.example .env
````

>  _Step 3. update OPENAI_API_KEY on .env file
>  _Step 4. Run this your terminal:

````
    composer install 
````

>  _Step 5.

````
    php artisan migrate
````

>  _Step 6.

````
    php artisan serve
````
>  _Step 8. Open the development server http://127.0.0.1:8000
